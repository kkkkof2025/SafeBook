# 成为白帽黑客之路

> 安全行业入门指南 — 你需要什么基础、学什么、往哪走

---

## 什么人可以学安全？

这是一个很现实的入门问题。答案是：**只要有兴趣和耐心，任何人**。

### 你不需要

- ❌ 不需要是计算机专业
- ❌ 不需要会写代码（可以先学）
- ❌ 不需要高学历
- ❌ 不需要英语很好（但学好英语会走得更远）

### 你需要

- ✅ 好奇心——想知道"为什么这样会出问题"
- ✅ 耐心——一个漏洞可能要反复测试几十次
- ✅ 动手能力——光看没用，要自己动手试
- ✅ 搜索能力——遇到了难题知道去哪里找答案

---

## 安全行业的角色

| 角色 | 做什么 | 需要什么基础 |
|------|--------|-------------|
| 渗透测试 | 模拟黑客攻击，找漏洞 | Web 基础、网络协议、工具使用 |
| 安全开发 | 写安全的代码、做代码审计 | 编程能力、常见漏洞原理 |
| 安全运维 | 维护安全设备、应急响应 | 网络基础、系统管理 |
| 安全研究 | 挖 0day、发论文 | 深厚的底层知识、逆向能力 |
| 安全运营 (SOC) | 监控告警、分析威胁 | 网络基础、日志分析 |

---

## 学习路线速览

```
基础入门（1-3 个月）
├── 计算机网络（TCP/IP、DNS、HTTP）
├── Web 应用架构（前后端、数据库）
├── Linux / Windows 基础
└── 安全工具入门（Burp Suite、nmap）

Web 安全（3-6 个月）
├── OWASP Top 10 逐个攻破
├── 靶场练习（DVWA、PortSwigger Labs）
└── 实战演练（CTF、Bug Bounty）

进阶方向（6-12 个月）
├── 内网渗透 / 域渗透
├── 移动安全
├── 云安全
├── 逆向工程
├── AI 安全
└── 代码审计
```

---

## 推荐学习顺序

1. **先建立基础**：理解网络怎么工作的、网页怎么渲染的
2. **动手实验**：搭建 DVWA 靶场，从 SQL 注入开始
3. **系统学习**：按 OWASP Top 10 逐个攻破
4. **实战练习**：PortSwigger Labs → CTF → 漏洞赏金
5. **深入方向**：选一个方向深耕
