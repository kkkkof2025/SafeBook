## 什么人可以学安全？

答案是：**只要有兴趣和耐心，任何人**。安全行业的人才缺口超过 300 万（ISC² 2023），现在是入行的最佳时机。

### 你不需要
- ❌ 不需要是计算机专业（行业里历史/法律/英语专业出身的很多）
- ❌ 不需要一开始就会写代码（边学边入门）
- ❌ 不需要高学历（能力 > 学历）
- ❌ 不需要英语母语水平（但阅读英文文档是必备技能）

### 你真正需要
- ✅ **好奇心** — 看到二维码想扫一下、看到登录框想试注入
- ✅ **耐心** — 一个漏洞可能要反复测试几十次
- ✅ **动手能力** — 光看教程没用，必须自己搭环境练习
- ✅ **搜索能力** — Google + GitHub + StackOverflow 是核心生产力

## 详细学习路线图

```
Phase 1: 基础（1-3 个月）
├── 计算机网络 TCP/IP、DNS、HTTP/HTTPS、子网划分
├── Web 应用架构：前后端交互、REST API、数据库
├── Linux 基础：命令行、文件权限、进程管理
├── 编程入门：Python（首选）+ Bash
└── 安全工具：Burp Suite、nmap、Wireshark

Phase 2: Web 安全（3-6 个月）
├── OWASP Top 10：逐个漏洞原理→检测→利用→修复
├── 靶场练习：DVWA → PortSwigger Labs → HackTheBox
├── 工具深入：Burp Suite Pro、sqlmap、ffuf、nuclei
└── 写第一份渗透测试报告

Phase 3: 进阶技能（6-12 个月）
├── 内网渗透：BloodHound、Mimikatz、横向移动
├── 代码审计：Semgrep、CodeQL、手动审计
├── 云安全：AWS/Azure 常见配置错误
├── 移动安全：Android APK 逆向、iOS 越狱检测
├── AI 安全：Prompt 注入、LLM 红队
└── 打 CTF：PicoCTF → 国内赛 → DEFCON Quals

Phase 4: 专业方向（12+ 个月）
├── 选择一个方向深耕
├── 考取行业认证
├── 发表安全研究/挖 CVE
└── 参与 Bug Bounty
```

## 推荐证书路径

| 阶段 | 证书 | 方向 | 费用 |
|------|------|------|------|
| 入门 | CompTIA Security+ | 信息安全基础 | $392 |
| 初级 | eJPT (INE) | 渗透测试 | $200 |
| 中级 | OSCP (OffSec) | 渗透测试 | $1599 |
| 高级 | OSEP / OSED | 逃逸/漏洞开发 | $1299+ |
| 管理 | CISSP (ISC²) | 安全管理 | $749 |
| 云 | AWS Security Specialty | 云安全 | $300 |
| 蓝队 | BTL1 / GCFA | 事件响应/取证 | $2000+ |

> **建议**：先考实战型证书（OSCP/eJPT），再考虑管理型证书（CISSP）。

## 三条常见成长路径

```
路径 A: 渗透测试专家
  CTF → Bug Bounty → 渗透测试工程师 → 红队 → APT 模拟

路径 B: 安全开发/架构
  安全编码 → SAST/DAST → DevSecOps → 安全架构师 → CSO

路径 C: 安全运营/蓝队
  SOC 分析师 → SIEM 工程师 → 威胁狩猎 → 应急响应专家
```
