## 为什么现在关注量子安全

```
量子计算机里程碑:
  2019 - Google 实现量子霸权 (53 qubits)
  2023 - IBM 发布 1000+ qubit 处理器
  203X - 预期可破解 RSA-2048 (估计)

威胁: Store-Now-Decrypt-Later
  → 攻击者现在窃取加密数据
  → 等待量子计算机成熟后解密
  → 你的数据今天就已经面临风险
```
