# SOC 安全运营

> 安全运营中心——企业安全的"神经中枢"

---

## SOC 架构

```
SOC 层次结构:

  L1 分析师   → 告警分类/初筛
  L2 分析师   → 事件调查/响应
  L3 分析师   → 威胁狩猎/高级分析
  SOC 经理    → 流程/KPI/报告
  CISO        → 战略/合规/预算
```

## 关键指标

| 指标 | 目标 |
|------|------|
| MTTD (平均检测时间) | < 1 小时 |
| MTTR (平均响应时间) | < 4 小时 |
| 误报率 | < 10% |
| 告警覆盖率 | > 95% |
