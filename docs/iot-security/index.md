# IoT 安全

> 200 亿设备，每一个都是一个攻击面

---

## IoT 攻击面

```
  IoT 设备
    ├── 硬件接口 (JTAG/UART/SPI)
    ├── 固件 (提取/逆向/修改)
    ├── 通信协议 (ZigBee/BLE/LoRa/MQTT)
    ├── 云端 API
    └── 移动 App
```
