# Web 安全概述

Web 安全是应用安全的基础，也是 OWASP Top 10（开放式 Web 应用程序安全项目十大安全风险）的核心关注领域。本章覆盖最常见的 Web 安全漏洞，每个漏洞独立成篇，包含原理、案例、POC 和修复方案。

- 按顺序读：从 SQL 注入开始，循序渐进
- 按需读：直接跳到感兴趣的漏洞
- 每一篇都是独立的，不需要依赖前文
