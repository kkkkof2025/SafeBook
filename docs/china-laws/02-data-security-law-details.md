# 数据安全法深度解读

## 概述

《中华人民共和国数据安全法》(DSL) 于 2021 年 9 月 1 日施行，它是中国数据安全领域的基础性法律。本章深入解读其关键条款和实操影响。

---

## 1. 核心制度

### 1.1 数据分类分级

```
数据分类分级制度:

  分类 (按来源/用途):
    ├── 个人数据 (个人信息)
    ├── 重要数据 (影响国家安全)
    ├── 核心数据 (直接影响国家安全)
    └── 一般数据 (其他)

  分级 (按危害程度):
    一级 (一般数据)
    二级 (重要数据) → 影响国家安全/经济运行
    三级 (核心数据) → 严重危害国家安全

行动: 组织必须对其数据进行分类分级
```

### 1.2 数据安全审查

```yaml
数据安全审查触发条件:
  - 数据处理活动影响或可能影响国家安全
  - 重要数据出境
  - 关键信息基础设施运营者采购网络产品和服务

审查内容:
  - 数据安全管理制度
  - 数据处理活动合规性
  - 数据出境安全性
  - 安全措施有效性
```

## 2. 数据出境管理

### 2.1 出境评估

```python
class DataExportAssessment:
    """数据出境安全评估"""

    def requires_export_assessment(self, data_type, volume, recipient_country):
        """
        判断是否需要安全评估
        参考: 《数据出境安全评估办法》(2022.9.1)
        """

        # 必须评估的情况
        if data_type == 'important_data':
            return True, '重要数据出境必须评估'

        if data_type == 'personal_info':
            # 处理 100 万人以上个人信息
            if volume >= 1_000_000:
                return True, '处理 >100万 个人信息出境必须评估'

            # 累计向境外提供 10 万人以上个人信息
            if volume >= 100_000:
                return True, '累计出境 >10万 个人信息出境必须评估'

            # 累计向境外提供 1 万人以上敏感个人信息
            if volume >= 10_000:
                return True, '敏感个人信息出境必须评估'

        return False, '无需强制评估'

    def assessment_items(self):
        """评估事项"""
        return [
            '数据出境的目的、范围、方式合法性',
            '数据出境对国家安全、公共利益、个人权益的影响',
            '接收方的数据保护能力和水平',
            '数据出境后的泄露、篡改、滥用风险',
            '数据安全保护责任义务'
        ]
```

### 2.2 标准合同 (SCCs)

```
数据出境三种合规路径:

  1. 安全评估 (网信办)
     → 适用: 重要数据 + CIIO + 大规模个人数据

  2. 标准合同 (SCCs)
     → 适用: 非重要数据、非 CIIO、中小规模

  3. 认证
     → 适用: 跨国公司内部数据转移
```

## 3. 处罚机制

```yaml
数据安全法处罚 (第 45 条):

  一般违法:
    - 责令改正
    - 警告
    - 罚款 5-50 万元
    - 直接责任人罚款 1-10 万元

  拒不改正或严重后果:
    - 罚款 50-200 万元
    - 暂停业务/停业整顿
    - 吊销许可证

  特别严重:
    - 罚款 200-1000 万元
    - 吊销许可证

  核心数据违规:
    - 最高罚款 1000 万元
    - 追究刑事责任
```
