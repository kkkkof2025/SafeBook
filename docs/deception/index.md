# 欺骗防御

> 用攻击者的思路来防御——设置陷阱，主动收集情报

---

## 欺骗防御理念

```
传统防御: 防火墙 → IDS → SIEM → 人工分析 (被动)
欺骗防御: 诱饵 → 吸引攻击者 → 自动告警 → 情报产出 (主动)

蜜罐优势: 0 误报 — 正常用户永远不会访问蜜罐
```
