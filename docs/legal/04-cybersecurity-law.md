## 2. 核心义务

### 网络安全等级保护 (等保)
```yaml
等保 2.0 五个等级:
  第一级 (自主保护): 一般信息系统 → 备案
  第二级 (指导保护): 不涉密的重要系统 → 测评 (2年/次)
  第三级 (监督保护): 涉密/涉及国计民生 → 测评 (1年/次)
  第四级 (强制保护): 国家重要领域 → 测评 (半年/次)
  第五级 (专控保护): 国家核心机密 → 专门保护

等保技术维度:
  - 安全物理环境 (机房/防火/温控)
  - 安全通信网络 (网络架构/加密传输)
  - 安全区域边界 (防火墙/IDS/边界防护)
  - 安全计算环境 (服务器/数据库/应用)
  - 安全管理中心 (日志/审计/态势感知)
```

### 数据本地化
```yaml
数据跨境传输要求:
  CIIO (关键信息基础设施运营者):
    - 个人信息和重要数据必须境内存储
    - 确需出境的,须通过安全评估

  非 CIIO:
    - 个人信息出境可选路径:
      1. 网信办安全评估
      2. 专业机构个人信息保护认证
      3. 标准合同 (SCC)
      4. 其他法定条件
```

## 3. 合规实施路径

### 网络运营者安全义务
```python
class CyberSecurityLawCompliance:
    """网络安全法合规检查"""

    OBLIGATIONS = {
        '网络安全等级保护': [
            '完成等保定级备案',
            '定期安全测评 (2级: 2年/次, 3级: 1年/次)',
            '安全整改闭环',
        ],
        '个人信息保护': [
            '制定并公示隐私政策',
            '收集个人信息需告知并取得同意',
            '提供删除权/更正权',
            '建立个人信息保护负责人制度',
        ],
        '网络安全事件': [
            '制定网络安全事件应急预案',
            '重大事件报告: 向网信办+公安',
            '至少每 6 个月应急演练一次',
        ],
        '技术保障': [
            '日志留存: ≥6个月',
            '数据分类分级 + 加密',
            '安全审计: 操作记录不可删除',
        ],
    }
```

### 处罚力度
| 违法行为 | 罚款 | 其他措施 |
|---------|------|---------|
| 不履行安全保护义务 | ¥1-10 万 (个人 ¥0.5-5 万) | 责令改正 |
| 拒不改正/导致危害 | ¥5-50 万 | 暂停业务/关闭网站 |
| 侵害个人信息 | ¥1-100 万 | 吊销许可证 |
| 情节严重 | ¥5000 万/5%营收 | 停业整顿/刑事追责 |

## 4. 与 GDPR 对比

| 维度 | 网络安全法/PIPL | GDPR |
|------|----------------|------|
| 管辖范围 | 中国境内 + 域外数据处理 | 欧盟境内 + 域外管辖 |
| 同意机制 | 告知-同意 (强同意) | 6种合法性基础 |
| 数据本地化 | CIIO 强制本地化 | 无强制要求 |
| 跨境传输 | 安全评估/认证/SCC | 充分性认定/SCC/BCR |
| 处罚上限 | ¥5000万/5%营收 | €2000万/4%营收 |
| DPO | 中国境内代表 (≥100万人信息) | 强制 DPO |
| 数据泄露 | 立即通知 | 72小时 |

## 5. 合规检查清单

```yaml
网络安全法合规自查:

  基础合规:
    - [ ] 等保定级备案完成
    - [ ] 网络安全负责人任命
    - [ ] 网络安全管理制度文档
    - [ ] 用户实名认证 (中国手机号)

  技术合规:
    - [ ] 日志留存 ≥ 6个月
    - [ ] 数据加密存储 + 传输
    - [ ] 入侵检测/防御系统
    - [ ] 漏洞扫描 (合规要求的周期)
    - [ ] 安全事件监控 + 告警

  隐私合规:
    - [ ] 隐私政策 (中文, 清晰, 易访问)
    - [ ] 收集前取得同意 (弹窗/勾选)
    - [ ] 用户数据删除功能
    - [ ] 未成年人信息特殊保护 (<14岁需监护人同意)

  跨境传输:
    - [ ] 数据出境安全评估 (如适用)
    - [ ] 标准合同签署 (如适用)
    - [ ] 数据本地化存储 (CIIO 必须)
```
