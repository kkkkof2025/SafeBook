# 威胁情报

> 知道敌人是谁、用什么工具、怎么攻击——知己知彼，百战不殆

---

## 威胁情报金字塔

```
威胁情报层次:

  战略情报 ─→ 给 CISO/董事会 (趋势/地缘政治/行业)
  战术情报 ─→ 给安全团队 (TTP/MITRE ATT&CK)
  运营情报 ─→ 给 SOC (IOC/告警/规则)
  技术情报 ─→ 给安全工具 (IP/Hash/域名/URL)
```
