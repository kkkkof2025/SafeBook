# HVV 蓝队防御策略

## 防御体系架构

### 纵深防御四层
`
边界防御 → 网络防御 → 主机防御 → 应用防御
   │           │           │           │
   WAF         IDS        EDR       RASP
   CDN         NDR        HIDS       IAST
`

## 关键发现阶段（0-6 小时）

### 异常检测指标
| 指标 | 告警阈值 | 工具 |
|------|---------|------|
| 登录失败率 | >5/min | Wazuh/SIEM |
| DNS 请求异常域 | >50/10min | Zeek |
| 横向连接新建 | >100/min | NDR |
| 进程执行链异常 | - | EDR |

### 应急响应 SOP
`
1. 确认告警     → 分析原始日志
2. 定性定级     → 高危/中危/低危
3. 启动响应     → 执行预定义剧本
4. 彻底清除     → 杀进程、删文件、封IP
5. 溯源反制     → 蜜罐、日志追溯
`

## 实战对抗技巧

- **蜜罐部署**：每段 VLAN 至少 2 台蜜罐
- **日志不可篡改**：Wazuh + SIEM 异地备份
- **最小权限**：Jump Server + 双因子认证
- **0day 应急**：虚拟补丁 + 入口流量全检测
- **钓鱼演练**：每月至少 1 次全员钓鱼测试
