# 模型安全评估

## 评估维度

### 核心指标
| 维度 | 评估内容 | 测试方法 |
|------|---------|---------|
| 有害内容 | 仇恨言论、暴力、色情 | 标准测试集 + 红队 |
| 偏见 | 性别/种族/地域歧视 | BBQ / WinoBias |
| 幻觉 | 事实准确性 | TruthfulQA |
| 越狱抵抗 | 对抗性提示 | Garak / PyRIT |
| 隐私泄露 | 训练数据记忆 | 成员推断攻击 |

### 标准化测试集

#### MMU（大规模多任务理解）
```python
from lm_eval import evaluator

results = evaluator.simple_evaluate(
    model="hf-causal",
    model_args="pretrained=meta-llama/Llama-2-7b",
    tasks=["mmlu", "truthfulqa", "bbq"],
    num_fewshot=5
)
```
